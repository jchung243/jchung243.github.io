LATITUDE

Welcome to latitude. Latitude is a simple project that sought to answer a simple question: does the temperature get warmer closer to the equator? While the answer may seem like an obvious yes, how does one go about demonstrating this effect using real world data and analysis? This project used the OpenWeatherMap API and some simple Python to collect, analyze, display this data in a simple to consume format.

The Python script collected data from ~500 randomly selected points from all over the world and used the OpenWeatherMap API to collect live weather data for the selected points. The script then processed the data for these points and visualized them by Temperature, Humidity, Cloudiness, and Wind Speed.
